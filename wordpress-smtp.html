WP不能正常发邮件解决方案

发表于 2017年03月24日 | 分类于 日记 |

用wordpress搭建的博客在不同主机下可能会遇到不能正常发邮件的问题。博主最近也遇到了这个问题，网上搜索了半天各种办法都用尽了还是没有解决问题o(╯□╰)o…正要放弃的时候突然想到是不是邮箱的问题啊，于是换了几个邮箱终于把这个问题解决了。

首先，如果你的主机不支持main()函数啥的那就不折腾果断用smtp发邮件，我用的是wp-mail-smtp 这个插件。
邮箱选择：博主测试了QQ邮箱、Hotmail邮箱…都不行，只有163邮箱没出现问题。

插件的设置是这样的：
From Email 这里的邮箱和 Username 这里的要一致。(切记！)
From Name 随便填； Mailer 选择第一项
Return Path 我没填
SMTP Host smtp.163.com
SMTP Port 25 Encryption 选择第一项 No encryption.
Authentication 选择第二项（切记！）
Username 你的邮箱同From Email
Password 邮箱密码

坚持原创分享，随意就好！
